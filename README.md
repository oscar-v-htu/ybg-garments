# YBG Garments - Premium Bespoke Tailoring

A high-end web landing page for **YBG Garments**, a bespoke tailoring brand based in Accra, Ghana. This project features a dark-themed, luxury aesthetic designed to showcase traditional and casual African attire.

## Features
* **Dynamic Gallery**: Showcases Traditional and Casual collections with localized pricing (GH₵).
* **Measurement Guide**: Integrated instructions for customer self-measurement.
* **WhatsApp Integration**: Direct "Order on WhatsApp" functionality for seamless customer conversion.
* **PWA Ready**: Includes a manifest and service worker for a mobile-app-like experience.

## Technical Details
* **Developed for**: HTU Faculty of Applied Sciences.
* **Technologies**: HTML5, CSS3 (Flexbox), and Progressive Web App (PWA) basics.
